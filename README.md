# 5 завдання. Гра-блукачка
Яка суть гри?
Тобі потрібно знайти предмети і здогадатися, якими з них можна побороти того чи іншого ворога.
Ти можеш заходити у такі кімнати:
<ul>
  <li>Kitchen</li>
  <li>Dining room</li>
  <li>Ballroom</li>
</ul>
У кухні ти можеш зустріти друга - фею. Якщо ти поговориш з нею, то отримаєш підказки. Для цього просто напиши команду <strong>"talk"</strong>.
У всіх інших кімнатах ти зустрінеш ворогів: павука та зомбі. З ними ти також можеш поговорити, але головна їхня відмінність від друзів - це те, що ти можеш з ними битися. Для цього напиши команду  <strong>"fight"</strong> і після того предмет, яким хочеш скористатися (з тих, які ти взяв). 
Щоб взяти предмет, просто напиши команду  <strong>"take"</strong>
Якщо ти хочеш переміститися в іншу кімнату, то впиши напрям, який тобі потрібен із запропонованих грою.
Коли ти використаєш усі предмети, тобто переможеш своїх ворогів, або застосуєш предмет не за призначенням, то гра закінчиться.
Підказки: щоб побороти ворога Dave - використай сир ("cheese"), Tabitha - книжка ("book")

# 6 завдання. Гра "Ярич"
Яка суть гри?
Гра показує день життя школяра у Новому Яричеві - містечку, яке знаходиться неподалік від Львова. Ти повинен вирішити усі проблеми головного героя, щоб виграти гру. Кожен з так званих ворогів потребує виконання якогось завдання, для цього тобі потрібно знайти необхідні предмети.
У тебе є можливість ходити у такі місця:
<ul>
  <li>Хата - вирішити проблему з Мамочкою</li>
  <li>Кімната Павла - попросити у брата допомоги</li>
  <li>Школа - вирішити проблему з Жанною Олександрівною (вчителькою)</li>
  <li>Учительська - попросити у Мамусі допомоги</li>
  <li>Музична школа - вирішити проблему з Дідом</li>
  <li>Народний дім - вирішити проблему з Наталею Іванівною (головна в ОТГ)</li>
</ul>
На кожній з локацій ти можеш поговорити з персонажем (команда "поговорити"), взяти предмет (команда "взяти"). Також є можливість вирішувати проблеми з ворогом (команда "вирішувати") завдяки предметам, які ти зібрав. У кімнатах з другом ти можеш попросити допомоги (команда "допомогти"), тоді ти отримаєш предмет, який тобі допоможе у подальшому. У цих місцях ти можеш скористатися підказкою (команда "підказати"). Щоб переміститися у інше місце, обери те, яке хочеш, із запропонованих у грі. Виграєш ти тоді, коли зможеш вирішити усі проблеми, якщо неправильно застосуєш предмет - програєш.
Підказки: Мамочка - щоденник (взяти у Павла), Дідо - грамота, Жанна Олександрівна - твір (взяти у Мамусі), Наталя Іванівна - звіт.
